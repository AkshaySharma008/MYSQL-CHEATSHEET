# MYSQL-CHEATSHEET

#### 1. Login to MySQL 
      a. mysql -u root -p
      
#### 2. Quit
      a. exit;
      
#### 3. Display all DB
      a. show databases;
      
#### 4. Create a database
      a. create database mytest;
      
#### 5. To use mytest database
      a. use mytest;
      
#### 6. Delete a database
      a. DROP DATABASE mytest;
      
#### 7. To get the version of mysql
      a. select version();
 
#### 8. To get the present date
      a. select now();
